Detailed Description

Class for system state monitoring and alarm generation.

ALARM performs continuous monitoring of critical system parameters and generates alarms when dangerous conditions are detected. Supports three levels of alarm criticality:

  • Warning: Warnings about approaching critical values
  • Error: Errors when exceeding safe ranges
  • Critical: Critical errors requiring immediate intervention

Monitoring includes:

  • Equipment temperature
  • Output power
  • Input power
  • Deviations from nominal values
